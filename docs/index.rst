Z-Rad Documentation
===================

Z-Rad is a radiomics toolkit with both a graphical interface and a Python API.
The documentation is organized by audience: end users start in the user guide,
developers use the reference and contributor sections, and project-specific
material such as IBSI validation lives separately.
